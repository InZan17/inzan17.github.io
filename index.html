About the launcher

This launcher offers a way to download and manage mods, modpacks and several Minecraft instances.

This launcher doesn't use HTML and CSS for the frontend, or any native UI. Instead it uses Miniquad to render UI, and Lua (or more specifically, Luau) to control the UI.

It's able to load custom Lua code the user has made, so users will be able to make their own frontend with this launcher. The aim is to make the API as simple and as customizable as possible.
